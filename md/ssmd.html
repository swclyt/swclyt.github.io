>用的开发工具是MyEclipse，项目是web-project - zookeeper注册中心 + tomcat + dubbo-admin - provider 服务提供者(spring + druid + mybatis + dubbo + zookeeper) - consumer 服务消费者(spring + spring mvc + dubbo + zookeeper) ### zookeeper注册中心 >这里用的是 zookeeper-3.4.6 解压到部署目录，找到`conf`文件夹进行配置，复制` zoo_sample.cfg`，改名为` zoo.cfg` 修改如下位置，设置好dataDir目录 ``` # example sakes. dataDir=D:/zookeeper-3.4.6/data ``` 可以额外添加dataLogDir目录，在dataDir下添加一行 ``` dateLogDir=D:/zookeeper-3.4.6/logs ``` 端口默认是2181，记得打开端口访问限制 ``` # the port at which the clients will connect clientPort=2181 ``` - 集群模式 使用集群模式再另外开
